# React Todo App

A simple and responsive Todo List application built using React.js.

## Features

* Add Todos
* Delete Todos
* React Router Navigation
* Bootstrap Styling
* Local Storage Support

## Tech Stack

* React.js
* JavaScript
* Bootstrap
* React Router DOM

## Installation

Clone the repository:

```bash
git clone https://github.com/kushagrashh/react-app.git
```

Move into the project directory:

```bash
cd react-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
